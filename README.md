# 🏨 Hotel Booking Web App (MERN Stack)

A full-stack **Hotel Booking Web Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** with secure authentication via **Clerk** and **Email Booking Confirmation**.

This application allows users to browse hotels, check availability, book rooms, and receive confirmation emails — all in a seamless and user-friendly interface.

---

## 🚀 Features

### 👤 User Features

* 🔐 Secure Authentication using **Clerk**
* 🏨 Browse hotels with details and images
* 📅 Check room availability
* 🛒 Book rooms online
* 📧 Receive booking confirmation via email
* 📜 View booking history

### 🛠️ Admin Features (if implemented)

* ➕ Add / Update / Delete hotels
* 📊 Manage bookings
* 👥 Manage users
